# Lakshay Kathpalia — Portfolio

A modern, animated personal portfolio built with **Next.js 15**, **Tailwind CSS**, and **Framer Motion**.

## ✨ Features

- Animated aurora background with floating gradient blobs
- Custom spotlight cursor (desktop)
- Scroll-progress bar + reveal-on-scroll animations
- Rotating role headline in the hero
- 3D tilt project cards
- Animated skill marquee
- Experience & education timelines
- Fully responsive + reduced-motion friendly

## 🚀 Getting started

```bash
npm install
npm run dev      # http://localhost:3000
```

Build for production:

```bash
npm run build
npm run start
```

## ☁️ Deploy to Vercel

1. Push this repo to GitHub.
2. Go to [vercel.com/new](https://vercel.com/new) and import the repo.
3. Vercel auto-detects Next.js — just click **Deploy**. No config needed.

Or use the CLI:

```bash
npm i -g vercel
vercel          # preview deploy
vercel --prod   # production deploy
```

## 🛠 Editing your content

All text/data lives in [`lib/data.ts`](lib/data.ts) — update your projects,
experience, skills, links, and profile there. Update social links (GitHub /
LinkedIn) in the `profile.socials` array.
